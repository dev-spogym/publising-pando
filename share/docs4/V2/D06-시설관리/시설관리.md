## SCR-050 락커 관리

> 유형: 화면
> URL 경로: `/locker`
> 연결 기능: FAC-01
> 연결 다이얼로그: DLG-050-001 / `docs4/V2/D06-시설관리/시설관리.md:866` 락커기록, DLG-050-002 / `docs4/V2/D06-시설관리/시설관리.md:915` 락커이동, DLG-050-003 / `docs4/V2/D06-시설관리/시설관리.md:962` 락커회수확인, DLG-050-004 / `docs4/V2/D06-시설관리/시설관리.md:1009` 개별배정, DLG-050-005 / `docs4/V2/D06-시설관리/시설관리.md:1056` 고장토글확인, DLG-050-006 / `docs4/V2/D06-시설관리/시설관리.md:1101` 일괄배정, DLG-050-007 / `docs4/V2/D06-시설관리/시설관리.md:1146` 일괄해제확인

### 1. 목적

센터 내 모든 락커의 배정 현황과 상태를 한눈에 파악하고 관리하는 화면입니다. 구역별(A/B/C) 그리드 배치도로 각 락커의 사용 여부·만료 임박·고장 상태를 시각적으로 확인하고, 회원 배정·이동·회수·고장 처리를 즉시 수행할 수 있습니다. 만료 임박 락커를 한 번에 일괄 해제하거나 여러 빈 락커를 한 명의 회원에게 일괄 배정하는 기능도 지원합니다. 자동 회수의 ON/OFF와 기본값은 D06이 아니라 D09 SCR-080A / `docs4/V2/D09-설정관리/설정관리.md:37` 지점 자동화 적용의 락커·사물함 자동 회수 정책을 따릅니다.

### 2. UI 구성

  - 사용중: 파란색 계열 / 만료 임박(HQ-09 / `docs4/V2/D10-본사관리/본사관리.md:710` 락커 만료 본사 step 또는 지점 추가 step 대상): 주황색 계열 / 고장: 빨간색 계열 / 빈 락커: 회색

- **일괄 배정**: 여러 빈 락커를 선택한 뒤 한 명의 회원에게 동시에 배정합니다.

- **엑셀 다운로드**: 현재 락커 전체 현황을 엑셀 파일로 내려받습니다.

### 3. 액션 / 흐름

6. 사용중 락커: [회수] 버튼 클릭 → 확인 다이얼로그 → 회수 완료.

7. 만료 임박 락커 일괄 처리: [만료 임박 일괄 해제] 클릭 → 해당 락커들을 클릭해 선택 → [일괄 해제 실행] → 확인 다이얼로그 → 완료.

## SCR-051 사물함 배정 관리

> 유형: 화면
> URL 경로: `/locker/management`
> 연결 기능: FAC-02

### 1. 목적
센터 내 일일 사물함·개인 사물함·골프 사물함의 실시간 이용 현황을 시각적 그리드로 보여주고, 회원을 검색해 빈 사물함에 즉시 배정할 수 있는 화면입니다. 일반 락커 관리(SCR-050 / `docs4/V2/D06-시설관리/시설관리.md:41`)가 장기 배정 관리에 초점을 맞춘다면, 이 화면은 당일 또는 단기 보관용 사물함(개인 소지품 보관)의 빠른 배정·해제에 초점을 맞춥니다.

동일한 `/locker/management` 화면 안에서 D11 SCR-I005 / `docs4/V2/D11-통합운영/통합운영.md:130` 고정 물품 락커 관리가 별도 탭 또는 섹션으로 함께 표시됩니다. SCR-051은 사물함 배정 운영 탭, SCR-I005는 상품 계약 기반 고정 물품 락커 운영 탭으로 구분합니다.

### 2. UI 구성
- **현황 요약 카드**: 총 사물함 수 / 사용 가능 수 / 사용중 수 / 시간 초과 수 / 상태 비정상 수를 5개 카드로 표시합니다.
- **자동 회수 정책 상태 배지**: D09 SCR-080A / `docs4/V2/D09-설정관리/설정관리.md:37` 지점 자동화 적용의 일일 사물함 22:00 상태 갱신(고정 ON)과 실제 일괄 회수 ON/OFF 상태를 읽기 전용으로 표시합니다.
- **사물함 유형 탭**: 일일 사물함 · 개인 사물함 · 골프 사물함으로 탭을 분리합니다. 각 탭에 현재 사물함 수를 표시합니다.
- **회원 배정 패널**: 탭 영역 상단에 배치됩니다.
  - 회원 검색 자동완성: 회원명 · 회원번호 · 연락처로 검색해 선택합니다.
  - 추천 사물함: 사용 가능한 사물함 중 번호가 가장 작은 것을 자동 추천합니다.
  - 만료일 입력: 배정 기간을 설정합니다 (기본값: 오늘로부터 3개월 후).
  - [배정하기] 버튼: 회원과 사물함이 모두 선택된 경우에만 활성화됩니다.
- **사물함 그리드**: 현재 탭의 사물함 전체를 격자로 표시합니다.
  - 빈 사물함: 클릭하면 해당 사물함이 배정 대상으로 선택됩니다.
  - 사용중: 파란색 / 시간 초과: 경고 아이콘 + 주황색 / 상태 비정상: 빨간색 (클릭 불가).
- **만료 사물함 일괄 해제**: 시간 초과(overtime) 상태 사물함을 한 번에 해제합니다.

### 3. 액션 / 흐름
1. 화면 진입 시 일일 사물함 탭이 기본 표시됩니다.
2. 회원 검색창에 이름·번호·연락처를 입력하면 자동완성 목록이 나타나고 회원을 선택합니다.
3. 추천 사물함이 자동 표시되거나, 그리드에서 원하는 빈 사물함을 직접 클릭해 선택합니다.
4. 만료일을 확인·수정한 뒤 [배정하기]를 클릭하면 배정 완료 메시지가 3초간 표시됩니다.
5. 시간 초과 사물함이 있을 경우 [만료 사물함 일괄 해제] 버튼이 활성화되어 한 번에 해제할 수 있습니다.

## SCR-052 밴드/카드 관리

> 유형: 화면
> URL 경로: `/rfid`
> 연결 기능: FAC-03
> 연결 다이얼로그: DLG-052-001 / `docs4/V2/D06-시설관리/시설관리.md:1193` RFID등록수정, DLG-052-002 / `docs4/V2/D06-시설관리/시설관리.md:1274` RFID이력조회, DLG-052-003 / `docs4/V2/D06-시설관리/시설관리.md:1351` RFID삭제확인

### 1. 목적
센터에서 사용하는 RFID 밴드와 카드를 등록·관리하고 회원 또는 직원과 연결하는 화면입니다. 회원이 락커·출입문·시설을 이용할 때 사용하는 RF 장치의 전체 발급 현황을 관리하며, 분실 신고·교체·삭제 처리를 수행할 수 있습니다.

### 2. UI 구성
- **현황 요약 카드**: 전체 카드 수 / 활성(사용중) 수 / 미사용·해제·분실 합계 수를 3개 카드로 표시합니다.
- **검색 및 상태 필터**: 카드 번호 또는 회원명으로 검색하고, 상태(활성/분실/해제)로 필터링합니다.
- **카드 목록 테이블**: 등록된 모든 RFID 카드·밴드의 목록을 표시합니다.
  - 표시 항목: 순번 / 등록일 / 카드 번호 / 연결 회원명 / 연락처 / 사용자 유형(회원·직원) / 상태 / 배정일 / 연결 사물함
  - 회원명 클릭 시 해당 회원 상세 화면으로 이동합니다.
  - 각 행에 이력 보기 · 분실 처리 · 수정 · 삭제 버튼이 있습니다.
- **신규 등록 버튼**: RFID 카드·밴드를 새로 등록합니다.

### 3. 액션 / 흐름
1. 화면 진입 시 전체 카드 목록이 표시됩니다.
2. 검색창에 카드 번호 또는 회원명을 입력하거나 상태 필터를 선택해 원하는 카드를 찾습니다.
3. **신규 등록**: [신규 등록] 버튼 클릭 → 카드 번호 스캔 또는 수동 입력 → 사용자 유형 선택 → 회원 검색 연결 → 연결 사물함 입력(선택) → 저장.
4. **분실 처리**: 해당 카드 행의 [분실 처리] 버튼 클릭 → 상태가 "분실"로 변경됩니다.
5. **이력 조회**: [이력] 버튼 클릭 → 기간을 선택해 해당 카드의 사용 이력을 조회합니다.
6. **삭제**: [삭제] 버튼 클릭 → 삭제 확인 다이얼로그 → 확인 시 영구 삭제.

## SCR-053 운동룸 관리

> 유형: 화면
> URL 경로: `/rooms`
> 연결 기능: FAC-04
> 연결 다이얼로그: DLG-053-001 / `docs4/V2/D06-시설관리/시설관리.md:1427` 룸등록수정, DLG-053-002 / `docs4/V2/D06-시설관리/시설관리.md:1505` 룸삭제확인

### 1. 목적
센터 내 운동룸(GX룸·PT룸·스피닝룸·필라테스룸 등)의 현황과 예약 슬롯을 관리하는 화면입니다. 각 룸의 운영 상태를 확인하고, 오늘의 시간대별 예약 혼잡도를 시각적으로 파악할 수 있습니다. 새 운동룸을 등록하거나 기존 룸 정보를 수정·삭제하고, 운영 상태를 빠르게 전환할 수 있습니다.

SCR-053은 독립 메뉴가 아니라 `/rooms`의 메인 화면인 SCR-059 / `docs4/V2/D06-시설관리/시설관리.md:789` 공간 자산 관리 안에 포함되는 운동룸 탭/섹션입니다. 운동룸 탭의 상세 필드, 카드, 등록/수정/삭제 동작은 본 섹션(SCR-053 / `docs4/V2/D06-시설관리/시설관리.md:264`)을 기준으로 구현합니다.

### 2. UI 구성
- **현황 요약 카드**: 전체 룸 수 / 운영중 수 / 점검·고장·미사용 합계 수 / GX룸 수·PT룸 수를 4개 카드로 표시합니다.
- **뷰 전환 탭**: 카드 보기와 목록 보기를 전환합니다.
- **유형 필터**: 전체 · GX · PT · 스피닝 · 필라테스 · 기타로 필터링합니다.
- **카드 보기**: 유형별로 묶어 각 룸을 카드 형태로 표시합니다.
  - 카드 내용: 룸명 / 유형 배지 / 수용 인원 / 게이트 / 운영 상태 배지 / 룸 설명 / 오늘 시간대별 예약 현황 슬롯 바 / 수정·삭제 버튼.
  - 슬롯 바: 07:00~20:00 시간대를 색상으로 표시합니다 (여유: 초록 / 혼잡: 주황 / 만석: 빨강 / 빈 시간: 회색).
  - 운영 상태 전환 버튼(⚙): 운영중 → 점검중 → 고장 → 운영중 순서로 순환 전환합니다.
- **목록 보기**: 룸 목록을 테이블 형태로 표시합니다.
- **새 운동룸 등록 버튼**: 새 운동룸을 등록합니다.

### 3. 액션 / 흐름
1. 화면 진입 시 전체 운동룸을 유형별로 묶어 카드 형태로 표시합니다.
2. 유형 필터를 선택하면 해당 유형의 룸만 표시됩니다.
3. **룸 등록**: [새 운동룸 등록] 클릭 → 룸명·유형·수용 인원·설명 입력 → 저장.
4. **룸 수정**: 카드의 [수정] 버튼 클릭 → 정보 수정 → 저장.
5. **룸 삭제**: 카드의 [삭제] 버튼 클릭 → 삭제 확인 다이얼로그 → 완료.

## SCR-054 골프 타석 관리

> 유형: 화면
> URL 경로: `/golf-bays`
> 연결 기능: FAC-05

### 1. 목적
골프 연습장 내 타석의 실시간 이용 현황과 예약 운영을 함께 관리하는 화면입니다. 타석별 사용자·프로·남은 이용 시간을 실시간으로 확인하고, 이용 시작·종료·타석 간 이동·점검 처리를 수행합니다. 또한 일간/월간 예약 캘린더, 개인레슨/정규/시설예약 구분, 예약 상세 처리 모달을 통해 골프 연습장 특화 운영 흐름을 한 화면군에서 다룹니다. 대기 중인 회원 목록과 자동 배정 기능으로 빈 타석이 생기면 다음 대기자에게 즉시 안내할 수 있습니다.

### 2. UI 구성
- **상단 운영 툴바**: 기간 헤더(`YYYY년 MM월 ~ YYYY년 MM월`), 이전/다음 이동, 오늘 바로가기, 보기 전환(일간/월간), 예약 구분 필터(전체 / 개인레슨 / 정규 / 시설예약)를 제공합니다.
- **예약 가용성 스트립**: 일간 보기에서 날짜 가로 스트립을 노출하고, 각 날짜 아래 타석별 예약 건수 또는 남은 슬롯 수를 작은 뱃지로 표시합니다.
- **현황 요약 카드**: 전체 타석 수 / 사용중 / 대기(빈 타석) / 예약 / 대기열 인원을 5개 카드로 표시합니다.
- **일간 타임테이블 뷰**: 가로축은 타석 번호, 세로축은 5분 단위 시간 슬롯으로 구성합니다.
  - 각 타석 헤더에 오늘 예약 건수/사용중 건수 뱃지를 함께 표시합니다.
  - 현재 시각은 붉은 가로선과 시각 라벨로 강조합니다.
  - 예약 블록은 유형별 색상(일반 시설예약 / 개인레슨 / 점검 블록)으로 구분하고, 블록 내부에 시간·회원명·프로 또는 예약 유형을 표시합니다.
  - 블록 클릭 시 예약 상세 모달이 열립니다.
- **월간 캘린더 뷰**: 날짜별로 타석 번호 또는 담당 프로 기준 예약 건수를 집계해 표시합니다.
  - 일자 셀 우상단에 총 예약 건수 배지를 표시합니다.
  - 셀 내부에는 타석/프로별 예약 건수 요약을 최대 3줄 노출하고, 초과분은 `+n` 또는 상세 패널로 확인합니다.
  - 날짜 클릭 시 해당 일자의 일간 타임테이블로 drill-down 됩니다.
- **실시간 타석 카드/오버레이**: 운영자는 타임테이블에서 사용중 타석의 현재 상태를 카드형 오버레이로 볼 수 있습니다.
  - 각 카드 내용: 타석 번호 / 상태 배지 / 이용 회원명 / 담당 프로명 / 남은 시간 카운트다운 / 프로젝터 on/off 토글 / 시작·종료·이동·점검 버튼.
  - 타석 상태: 대기(초록) / 사용중(파랑) / 예약(노랑) / 점검(회색).
  - 남은 시간이 0이 되면 알림이 표시됩니다.
- **예약 상세 모달**: 선택한 시간 블록의 날짜·시간·타석번호를 상단에 표시하고, 예약별 처리 버튼(삭제 / 취소 / 결석 / 완료)을 제공합니다.
  - 예약 카드에는 회원명, 연락처, 예약 유형(시설/개인레슨), 사용 상품, 남은 레슨 횟수, 모바일 알림 여부를 표시합니다.
  - 동일 시간대에 복수 예약이 걸린 예외 상황도 한 모달에서 일괄 처리할 수 있습니다.
- **대기열 패널**: 화면 하단에 현재 대기 중인 회원 목록을 순서대로 표시합니다.
  - 각 항목: 순서 번호 / 회원명 / 대기 시작 시간 / 예상 대기 시간.
  - 타석이 종료되면 첫 번째 대기자가 자동으로 해당 타석에 배정됩니다.

### 3. 액션 / 흐름
1. 화면 진입 시 전체 타석의 실시간 현황이 그리드로 표시됩니다.
2. 운영자는 일간/월간 보기와 예약 구분 필터를 전환해 현재 운영 문맥(실시간 현황, 특정 날짜 예약량, 개인레슨 집중 보기)을 맞춥니다.
3. **이용 시작**: 빈 타석의 [시작] 버튼 클릭 → 회원명 입력 → 이용 시작 (타이머 카운트다운 시작).
4. **예약 상세 확인**: 일간 타임테이블 또는 월간 캘린더의 예약 블록 클릭 → 예약 상세 모달 오픈 → 삭제/취소/결석/완료 중 처리.
5. **이용 종료**: 사용중 타석의 [종료] 버튼 클릭 → 타석 초기화 → 대기열 첫 번째 회원이 있으면 자동 배정.
6. **타석 이동**: 사용중 타석의 [이동] 버튼 클릭 → 이동할 대상 타석 선택 → 회원 정보와 남은 시간이 새 타석으로 이동.
7. **점검 처리**: 빈 타석의 [점검] 버튼 클릭 → 점검 상태로 전환. 점검중 타석의 [점검 해제]로 복구.

## SCR-055 상품 재고 관리

> 유형: 화면
> URL 경로: `/facility/inventory`
> 연결 기능: FAC-06

### 1. 목적
센터에서 실제로 보유한 일반 재고 물품의 수량을 관리하는 화면입니다. 관리 대상은 운동복, 수건, 세제, 소모품, 판매용 용품 등 센터가 실물로 보유하고 수량 확인이 필요한 품목입니다.

D05 상품관리의 `운동복` 상품은 회원에게 청구하는 운동복 대여비/상품 비용을 등록하는 결제용 상품입니다. D05에서 운동복 상품이 판매되거나 결제되어도 본 화면의 재고 수량은 자동 차감하지 않습니다. 실제 재고 수량 증감은 본 화면에서 입고, 출고, 조정으로만 처리합니다.

회원별 운동복 대여·반납, 세탁중, 파손, 복구 상태 관리는 v1 범위에서 사용하지 않습니다. 운동복도 본 화면에서는 일반 재고 품목 중 하나로만 관리합니다.

### 2. UI 구성
- **현황 요약 카드**: 전체 품목 수 / 정상 재고 품목 수 / 부족 재고 품목 수 / 품절 품목 수를 표시합니다.
- **상태 탭**: 전체 · 정상 · 부족 · 품절 탭으로 필터링하고 각 탭에 건수를 표시합니다.
- **검색창**: 품목명, 품목 코드, 분류명으로 검색합니다.
- **상품 재고 목록 테이블**: 등록된 재고 품목 전체를 표시합니다.
  - 표시 항목: 품목명 / 분류 / 현재 수량 / 안전 재고 / 재고 상태 / 최근 입고일 / 최근 출고일 / 담당자 / 액션.
  - 액션 버튼: [입고] [출고] [조정] [이력].
- **품목 등록 버튼**: 신규 재고 품목을 추가합니다.
- **엑셀 다운로드**: 현재 필터 결과 기준으로 재고 현황을 엑셀로 내려받습니다.

### 3. 액션 / 흐름
1. 화면 진입 시 전체 재고 품목 목록이 표시됩니다.
2. 상태 탭 또는 검색창을 이용해 원하는 품목을 찾습니다.
3. **입고 처리**: [입고] 클릭 → 입고 수량, 입고일, 단가, 거래처, 비고 입력 → 현재 수량 증가.
4. **출고 처리**: [출고] 클릭 → 출고 수량, 출고 사유, 출고일, 비고 입력 → 현재 수량 감소.
5. **재고 조정**: 실사 결과가 시스템 수량과 다를 때 [조정] 클릭 → 실제 수량과 조정 사유 입력 → 현재 수량 보정.
6. **품목 등록**: [품목 등록] 클릭 → 품목명, 분류, 단위, 안전 재고, 메모 입력 → 저장.

### 2. UI 구성

- **장비 목록 테이블**: 등록된 모든 장비를 표시합니다.

- **장비 등록 버튼**: 새 장비 항목을 추가합니다.
- **점검 등록 버튼**: 장비별 점검 완료 기록을 입력합니다.

### 1. 목적

### 2. UI 구성

## DLG-052-001 RFID 등록/수정

> 유형: 다이얼로그
> URL 경로: `/rfid`
> 연결 기능: FAC-03
> 연결 다이얼로그: DLG-052-001 / `docs4/V2/D06-시설관리/시설관리.md:1193` RFID등록수정, DLG-052-002 / `docs4/V2/D06-시설관리/시설관리.md:1274` RFID이력조회, DLG-052-003 / `docs4/V2/D06-시설관리/시설관리.md:1351` RFID삭제확인

### 1. 목적
RFID 카드 또는 밴드를 새로 등록하거나 기존 정보를 수정하는 다이얼로그입니다. 카드 번호를 스캔하거나 수동으로 입력하고, 연결할 회원 또는 직원을 지정합니다.

### 2. UI 구성
- **다이얼로그 제목**: "RFID 카드 등록" (신규) 또는 "RFID 카드 수정" (수정)
- **카드 번호 입력 영역**: 스캔 대기 상태를 시각적으로 표시합니다 (무선 아이콘 + 깜빡임 효과). 수동 입력도 가능합니다.
- **사용자 유형 선택**: 회원 / 직원 중 하나를 선택합니다.
- **회원 연결 자동완성**: 회원명 또는 회원 번호로 검색해 연결할 회원을 선택합니다.
- **연결 사물함 입력**: 해당 카드와 연결된 사물함 번호를 입력합니다 (선택 사항).
- **저장 버튼**: 입력한 정보를 저장합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

### 3. 액션 / 흐름
1. [신규 등록] 클릭 또는 기존 카드의 [수정] 클릭 → 다이얼로그 열림.
2. 카드 번호를 스캔하거나 직접 입력합니다.
3. 사용자 유형을 선택하고 회원을 검색해 연결합니다.
4. 사물함 번호를 입력(선택)한 뒤 [저장]을 클릭합니다.

### 4. 핵심 상태
| 상태 | 설명 |
|---|---|
| 신규 등록 | 모든 필드 빈 상태로 열림 |
| 수정 | 기존 카드 정보가 미리 채워진 상태로 열림 |
| 카드 번호 미입력 | [저장] 버튼 비활성화 |
| 저장 성공 | 다이얼로그 닫힘 + 목록 갱신 |

| 계정 | 동작 기준 |
|---|---|
| Owner(지점장) / manager | 부모 화면에서 허용된 경우 조회·입력·저장할 수 있습니다. |

## DLG-058-001 청소 스케줄 등록

> 유형: 다이얼로그
> URL 경로: `/cleaning-schedule`
> 연결 기능: FAC-EXT-04
> 연결 다이얼로그: DLG-058-001 / `docs4/V2/D06-시설관리/시설관리.md:2064` 청소스케줄등록

### 1. 목적
청소 구역·담당자·주기·예정 시간을 설정해 청소 일정을 등록하는 다이얼로그입니다. 등록된 일정은 이후 해당 주기마다 자동으로 체크리스트에 생성됩니다.

### 2. UI 구성
- **다이얼로그 제목**: "청소 스케줄 등록"
- **구역명 입력**: 청소할 구역 이름을 입력합니다 (예: 남자 탈의실, GX룸, 로비).
- **청소 유형 선택**: 일일 청소 / 주간 청소 / 월간 청소 중 선택합니다.
- **담당자 선택**: 청소를 담당할 직원을 선택합니다.
- **예정 시간 입력**: 청소 예정 시간을 입력합니다 (예: 08:00).
- **요일 선택** (주간 청소 선택 시): 해당 청소를 수행할 요일을 선택합니다.
- **일자 선택** (월간 청소 선택 시): 매월 몇 일에 수행할지 선택합니다.
- **시작일**: 이 스케줄을 적용하기 시작할 날짜를 입력합니다.
- **저장 버튼**: 청소 스케줄을 등록합니다.
- **취소 버튼**: 변경 없이 다이얼로그를 닫습니다.

### 3. 액션 / 흐름
1. [청소 스케줄 등록] 버튼 클릭 → 다이얼로그 열림.
2. 구역명·청소 유형·담당자·예정 시간을 입력합니다.
3. 주간/월간 선택 시 요일 또는 일자를 추가로 선택합니다.
4. [저장] 클릭 → 청소 일정 등록 + 체크리스트에 자동 반영.

### 4. 핵심 상태
| 상태 | 설명 |

| 주간 청소 선택 | 요일 선택 필드 표시 |
| 월간 청소 선택 | 일자 선택 필드 표시 |
