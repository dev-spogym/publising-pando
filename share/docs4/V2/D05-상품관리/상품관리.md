### D05 다이얼로그 ID 원칙
- D05에서는 `P001`처럼 번호만 축약해 부르지 않습니다. 항상 DLG-P001 / `docs4/V2/D05-상품관리/상품관리.md:705` 상품등록모달처럼 전체 ID를 사용합니다.
- 상품 등록과 상품 상세/수정은 실제 프론트 기준으로 같은 상품 패널 입력 UI를 공유합니다. 신규 등록은 `isNew=true` 상태, 기존 상품 조회/수정은 상품 데이터가 채워진 상태로 구분합니다.
- DLG-P001 / `docs4/V2/D05-상품관리/상품관리.md:705` 상품등록모달은 별도 컴포넌트명이 아니라, 공유 상품 패널의 신규 등록 상태를 가리키는 기획 식별자입니다.
- 별도 상품 수정 전용 모달은 사용하지 않습니다. 기존 상품 수정은 SCR-P003 / `docs4/V2/D05-상품관리/상품관리.md:206` 상품 상세/수정 패널에서 같은 패널 UI로 바로 처리합니다.
- 전 지점 배포부터 시즌가격 등록/수정까지 모든 다이얼로그는 P001~P023 번호대를 사용하되, 문서 참조 시에는 항상 DLG-P001 / `docs4/V2/D05-상품관리/상품관리.md:705` 상품등록모달 같은 전체 ID로 표기합니다.

## SCR-P002 상품 등록

> 유형: 화면/우측 패널
> URL 경로: `/products/new`
> 연결 기능: PRD-EXT-01
> 연결 다이얼로그: DLG-P001 / `docs4/V2/D05-상품관리/상품관리.md:705` 상품등록모달, DLG-P003 / `docs4/V1/D05-상품관리/상품관리.md:692` 작업취소확인, DLG-P008 / `docs4/V2/D05-상품관리/상품관리.md:828` 상품가져오기

### 1. 목적

신규 상품을 등록하는 화면입니다. SCR-P001 / `docs4/V2/D05-상품관리/상품관리.md:45` 상품 관리에서 [상품 등록]을 누르면 SCR-P003 / `docs4/V2/D05-상품관리/상품관리.md:206` 상품 상세/수정 패널과 같은 상품 패널 입력 UI가 신규 등록 상태로 열립니다. `/products/new` 직접 진입 시에도 동일 필드 구성을 가진 등록 화면으로 표시됩니다. 현재 프론트엔드의 공유 상품 패널 UI를 정본으로 합니다.

기존 상품 수정은 별도 상품 수정 모달을 열지 않고, 같은 상품 패널 UI에 기존 상품 값을 채운 SCR-P003 / `docs4/V2/D05-상품관리/상품관리.md:206` 상품 상세/수정 패널에서 바로 처리합니다.

## SCR-P004 할인 설정

> 유형: 화면
> URL 경로: `/discount-settings`
> 연결 기능: PRD-04

## SCR-P005 상품 카탈로그

> 유형: 화면
> URL 경로: `/products/catalog`
> 연결 기능: PRD-05

### 1. 목적
센터에서 판매 중인 상품을 고객이 보기 좋은 카탈로그 형태로 안내하는 화면입니다. 회원 상담 시 직원이 태블릿이나 모니터로 보여주거나, 인쇄·공유하여 외부 홍보에 활용할 수 있습니다.

### 2. UI 구성
### 카탈로그 뷰

- 상품 카드 그리드 레이아웃
- 각 카드: 상품명, 가격, 기간/횟수, 주요 혜택 요약
- 상품 대분류별 색상 구분(회원권, 수강권, 락커, 운동복, 일반)
- GX 상품은 요가 / 필라테스 / 스피닝 / 줌바 / GX 기타 세부종목 배지를 함께 표시
- 이미지 첨부 가능 (미구현)

### 필터 바

- 1단계 상품 대분류 탭 필터 (회원권 / 수강권 / 락커 / 운동복 / 일반)
- GX 탭 선택 시 2단계 세부종목 필터 표시 (요가 / 필라테스 / 스피닝 / 줌바 / GX 기타)
- 활성 상품만 표시 (비활성 상품 제외)

### 인쇄 · 공유 기능

- 인쇄: 브라우저 인쇄 다이얼로그 연동 (카탈로그 전용 레이아웃)
- 공유: 카탈로그 페이지 링크 복사

---

### 3. 액션 / 흐름
1. **카탈로그 조회**: 화면 진입 → 활성 상품이 카드 형태로 표시
2. **필터**: 상품 유형 탭 클릭 → 해당 유형 상품만 표시
3. **카탈로그 미리보기**: [DLG-P016 / `docs4/V2/D05-상품관리/상품관리.md:1162` 카탈로그미리보기]에서 고객에게 보여질 화면 확인
4. **옵션 설정**: [DLG-P017 / `docs4/V2/D05-상품관리/상품관리.md:1245` 카탈로그설정]에서 노출 옵션 설정
5. **내용 편집**: [DLG-P018 / `docs4/V2/D05-상품관리/상품관리.md:1335` 카탈로그편집]에서 카탈로그 문구·순서 편집
6. **인쇄**: 인쇄 버튼 클릭 → 카탈로그 전용 레이아웃으로 인쇄 다이얼로그 실행

## SCR-P006 상품 비교

> 유형: 화면
> URL 경로: `/products/compare`
> 연결 기능: PRD-06

### 1. 목적
최대 3개 상품을 나란히 비교하여 회원 상담 시 최적의 상품을 안내할 수 있는 화면입니다. 가격, 이용 기간, 포함 서비스, 옵션 항목을 동일한 기준으로 나란히 표시하여 상품 간 차이를 직관적으로 파악할 수 있습니다.

### 2. UI 구성

- 비교할 상품 최대 3개 선택 (드롭다운 또는 목록에서 선택)
- 선택된 상품은 각 열에 표시
- 상품 제거(X) 버튼으로 비교 대상에서 제외

| 홀딩 가능 | 일시정지 허용 여부 |
| 양도 가능 | 타인 양도 허용 여부 |
| 포인트 적립 | 포인트 적립 여부 |
| 키오스크 노출 | 키오스크 판매 여부 |

## DLG-P023-시즌가격등록수정 시즌 특가 등록/수정

> 유형: 다이얼로그
> 연결 기능: 없음 (DLG-P023는 docs4 내부에서 독립 보조 화면/다이얼로그로 관리)
> 연결 다이얼로그: DLG-P023 / `docs4/V2/D05-상품관리/상품관리.md:1797` 시즌가격등록수정

### 1. 목적
시즌 특별 가격을 등록하거나 기존 시즌 특가를 수정하는 모달입니다. 특정 기간 동안만 적용되는 할인 가격을 상품별로 설정할 수 있습니다.

### 2. UI 구성
| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 시즌 이름 | 필수 | 예: "2026 여름 특가", "신년 이벤트" |
| 적용 시작일 | 필수 | 특가 시작 날짜 |
| 적용 종료일 | 필수 | 특가 종료 날짜 (시작일 이후) |
| 대상 상품 선택 | 필수 | 체크박스로 특가 적용 상품 1개 이상 선택 |
| 특가 설정 방식 | 필수 | 정률(%) 할인 / 정액(원) 할인 / 직접 입력 가격 |
| 특가 값 | 필수 | 할인율, 할인 금액, 또는 직접 입력 가격 |
| 활성 상태 | 선택 | 기본값: 활성 |

하단 버튼: **취소** / **저장**

---

### 3. 액션 / 흐름
**등록 모드:**
1. 빈 폼으로 모달 열림
2. 시즌명, 기간, 대상 상품, 특가 설정 입력
3. "저장" 클릭
   - 필수 항목 미입력 시: 안내 메시지
   - 종료일이 시작일보다 이르면: "종료일을 확인하세요." 안내
   - 대상 상품 미선택 시: 안내 메시지
   - 정상 저장 시: 모달 닫힘 + 목록 갱신

**수정 모드:**
1. 기존 정보가 채워진 상태로 모달 열림
2. 원하는 항목 수정
3. "저장" 클릭 → 수정 저장 + 목록 갱신
4. "취소" 클릭: 변경 사항 폐기, 모달 닫힘

### 4. 핵심 상태
| 상태 | 설명 |

| 입력/수정 중 | 필수값 누락, 형식 오류, 권한 제한을 인라인 메시지로 표시합니다. |
