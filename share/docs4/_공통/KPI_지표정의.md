# KPI 지표 정의

## 1. 목적

이 문서는 KPI 관련 반복 문의를 줄이기 위한 공통 지표 정의입니다.

KPI는 별도 기능 묶음이 아니라 D10 본사관리의 대시보드, KPI 대시보드, KPI 센터, Today Tasks에서 표시·필터·정렬에 사용하는 운영 지표입니다. 화면 UI와 액션은 D10 본사관리 문서를 우선하며, 지표명과 산식의 공통 의미는 이 문서를 기준으로 확인합니다.

## 3. SCR-094 / `docs4/V1/D10-본사관리/본사관리.md:1212` KPI 대시보드 지표

> 연결 기능: HQ-02 / `docs4/V1/D10-본사관리/본사관리.md:1212`

SCR-094의 KPI 카드는 별도 계약 기능 여러 개가 아니라 하나의 KPI 대시보드 안에서 표시하는 카드 구성입니다. 모든 카드는 선택 기간과 지점 필터를 공통 적용합니다. 카드 클릭 시 외부 화면으로 자동 이동하지 않고 같은 화면의 관련 차트나 테이블 영역을 강조하거나 스크롤합니다.

1. 매출 목표 달성률: 기간 내 결제 확정 순매출을 설정된 월 매출 목표로 나눈 비율입니다. 목표 미설정 시 목표 미설정 배지와 목표 설정 버튼을 표시합니다.
2. 신규 회원 목표 달성률: 기간 내 첫 결제 완료 신규 회원 수를 신규 회원 목표로 나눈 비율입니다. 목표가 없으면 N/A로 표시합니다.
3. 출석률 목표 달성률: 예약 확정 수업·세션 중 출석 완료 건수를 출석 대상 건수로 나눈 비율입니다.
4. 회원 유지율 목표 달성률: 기간 종료 시점 활성 회원 수를 기간 시작 시점 회원 수로 나눈 비율입니다. 신규 등록 회원은 참고값으로만 표시합니다.
5. GX 이번 달 수업 건수: 기간 내 GX 수업 개설 건수입니다.
6. GX 예약 수: 기간 내 GX 수업 예약 확정 건수입니다. 예약 취소 건은 제외합니다.
7. GX 출석 수: 기간 내 GX 수업 출석 완료 건수입니다.
8. GX 수업 출석률: GX 출석 완료 건수를 GX 예약 확정 건수로 나눈 비율입니다.
9. PT 강습 세션수: 기간 내 완료된 PT 세션 수입니다.
10. GX 강습 세션수: 기간 내 완료된 GX 세션 수입니다.
11. 골프 강습 세션수: 기간 내 완료된 골프 세션 수입니다.
12. 기타 강습 세션수: 기간 내 완료된 기타 세션 수입니다.
13. WI/TI 문의 건수 비교: 방문 문의와 전화 문의 건수를 나란히 표시합니다.
14. TI 방문 전환율: 전화 문의 중 대면 상담으로 이어진 건수를 전화 문의 건수로 나눈 비율입니다.
15. TI 등록률: 전화 문의 중 결제 완료 등록 건수를 전화 문의 건수로 나눈 비율입니다.
16. WI 등록률: 방문 문의 중 결제 완료 등록 건수를 방문 문의 건수로 나눈 비율입니다.
17. 재등록 상담 완료 건수: 재등록 DB 상담결과가 등록, 확정, 예상, 관리, 실패 중 하나로 처리된 건수입니다.
18. 재등록 성공률: 재등록 DB의 등록 건수를 등록, 확정, 예상, 관리, 실패 전체 건수로 나눈 비율입니다.
19. 강습 OT 신규 배정: 기간 내 강습 OT 신규 배정 인원 또는 건수입니다.
20. 골프 OT 신규 배정: 기간 내 골프 OT 신규 배정 인원 또는 건수입니다.
21. OT 등록 전환율: OT 등록 건수를 실제 OT 진행 건수로 나눈 비율입니다.
22. OT 거부율: OT 거부와 보류 건수를 실제 OT 진행 건수로 나눈 비율입니다.
23. OT 이월률: OT 이월 건수를 OT 배정 인원으로 나눈 비율입니다.
24. OT 체험에서 등록률: OT 체험 대상 중 등록 완료 건수를 OT 체험 건수로 나눈 비율입니다.

## 4. 강습 세션 유형과 GX 세부종목

- 강습 세션 유형은 PT, GX, 골프, 기타입니다.
- 강습 세션 유형은 KPI 집계 축입니다.
- GX 세부종목은 요가, 필라테스, 스피닝, 줌바, GX 기타입니다.
- GX 세부종목은 상품·매출·수업 세부 분포 축입니다.
- 강습 세션 유형과 GX 세부종목을 같은 필터로 합치지 않습니다.

## 5. WI / TI 기준

- WI는 방문 문의입니다.
- TI는 전화 문의입니다.
- TI 방문 전환율은 전화 문의 후 실제 대면 상담으로 이어진 비율입니다.
- TI 등록률은 전화 문의 후 결제 완료 등록까지 이어진 비율입니다.
- WI 등록률은 방문 문의 후 결제 완료 등록까지 이어진 비율입니다.

## 6. 재등록 / 이탈 회복 KPI

- 재등록 성공률은 현재 이탈 정의가 확정되기 전까지 재등록률을 대체하는 지표입니다.
- 만료일별 고정 버킷 위젯은 사용하지 않습니다.
- HQ-09 / `docs4/V1/D10-본사관리/본사관리.md:1393` 회원 이용권 만료 본사 step 또는 지점 추가 step 대상자를 재등록 상담 대상 단일 업무로 표시합니다.

## 7. SCR-095 / `docs4/V1/D10-본사관리/본사관리.md:267` KPI 센터 역할별 보드

> 연결 기능: HQ-03 / `docs4/V1/D10-본사관리/본사관리.md:267`

- 본사 보드: 전 지점 지표, 지점/팀 성과 랭킹, 전사 운영 인사이트를 봅니다.
- 지점 보드: 소속 지점 핵심 지표, 재등록 상담 완료 건수, 재등록 성공률, 운영 액션 큐를 봅니다.
- FC 보드: 신규 리드, TI 방문 전환율, WI 등록률, TI 등록률, 상담 액션 큐를 봅니다.
- 트레이너 보드: OT 배정 합계, OT 등록 전환율, OT 체험에서 등록률, 트레이너별 OT 진행 현황을 봅니다.
- 운영 보드: 출석, 미수, 운영 액션, 일상 처리 업무를 봅니다.

## 8. 액션 큐 우선순위

SCR-095의 액션 큐는 오늘 처리해야 할 업무를 긴급, 우선, 일반, 완료가능 단계로 구분합니다.

- 긴급: 24시간 초과 미처리, 결제/환불/수업 운영상 즉시 대응하지 않으면 고객 불만이나 운영 손실이 발생하는 항목입니다.
- 우선: 오늘 안에 처리해야 하는 상담, 재등록, 미수, 만료 알림 대상입니다.
- 일반: 기한은 있으나 즉시 대응이 필요하지 않은 후속 확인 항목입니다.
- 완료가능: 조건이 충족되어 클릭 또는 확인만으로 완료 처리할 수 있는 항목입니다.

FC 보드의 기본 액션 큐는 전화 문의 후 미방문, 방문 후 미등록, 상담 결과 미기입, HQ-09 / `docs4/V1/D10-본사관리/본사관리.md:1393` 만료 알림 대상 재등록 상담입니다.

## 9. SCR-098 / `docs4/V1/D10-본사관리/본사관리.md:446` 오늘의 할 일 기준

> 연결 기능: HQ-06 / `docs4/V1/D10-본사관리/본사관리.md:446`

Today Tasks는 대시보드 알림 전체가 아니라 오늘 처리할 실행 업무 목록입니다.

- 필터 탭은 전체, 미완료, 완료입니다.
- 탭 배지는 오늘 날짜 기준 해당 상태의 태스크 건수입니다.
- 출처 배지는 수동, KPI, 일정 요청입니다.
- 일정 요청 미처리 건수는 수업 일정 요청 중 대기, 응답대기, 24시간 초과 미처리 건수를 기준으로 계산합니다.
- 24시간 초과 미처리 일정 요청은 긴급 태스크에 포함합니다.

## 10. 미확정 또는 후속 확인 항목

- 예측 분석, 벤치마크, 시나리오 비교처럼 통계 모델이 필요한 고도화 지표는 v1 개발 기준으로 바로 확정된 KPI가 아닙니다.
- 비용, 손익, 회계성 지표는 현재 문서상 v1 핵심 구현 기준으로 보지 않습니다.
- 고객사가 별도 랭킹 기준이나 점수 가중치를 확정하면 D10 본사관리와 이 문서를 함께 갱신합니다.
