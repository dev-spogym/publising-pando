# KIOSK / IoT 연동 매트릭스

> Health Connect / 회원 건강 연동 요약(SCR-I007 / `docs4/V1/D11-통합운영/통합운영.md:584`, IOT-05 / `docs4/V1/D11-통합운영/통합운영.md:470`)은 키오스크·IoT 장비 연동이 아니라 회원앱 MA-159(웹앱이라 정확하 경로기제X) 기반 Android Health Connect 조회 기능입니다. 키오스크/IoT 구현 범위로 해석하지 않고 `docs4/V1/D11-통합운영/통합운영.md:621`의 SCR-I007 / `docs4/V1/D11-통합운영/통합운영.md:584` 기준을 따릅니다.
> 키오스크 설정은 D09-설정관리(SCR-082 / `docs4/V1/D09-설정관리/설정관리.md:432`/SCR-082A / `docs4/V1/D09-설정관리/설정관리.md:526`/SCR-083 / `docs4/V1/D09-설정관리/설정관리.md:620`)에서만 관리합니다. SCR-082A와 SCR-083은 `/settings/iot` 같은 설정 페이지의 탭입니다. 키오스크 단말 운영 화면은 KIOSK 기획문서 범위이며, D11은 출석/락커/체성분 화면에서 기기 상태와 이벤트 수신값만 표시합니다.
> CRM에서 고정으로 사용하는 기기 종류는 출입 게이트 / 키오스크 / 락커 컨트롤러 / InBody 측정기 4종입니다.
