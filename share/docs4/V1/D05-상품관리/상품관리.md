# D05-상품관리 상품관리

> 개발사용 정리 문서입니다. 이 파일만으로 해당 도메인의 화면·다이얼로그 구현 기준을 확인할 수 있도록 재구성했습니다.




## 도메인 개요
- 화면/다이얼로그 구현 시 이 파일을 먼저 확인합니다.
- 본 문서가 해당 도메인 화면·다이얼로그 구현 기준입니다. 구현 판단을 위해 다른 폴더를 확인하지 않습니다.
- API/DB 상세는 백엔드 설계 범위이므로 제외하고, 화면/기능/UX/운영정책 기준만 남겼습니다.

### D05 다이얼로그 ID 원칙
- D05에서는 `P001`처럼 번호만 축약해 부르지 않습니다. 항상 DLG-P001 / `docs4/V1/D05-상품관리/상품관리.md:496` 상품등록모달처럼 전체 ID를 사용합니다.
- 상품 등록과 상품 상세/수정은 실제 프론트 기준으로 같은 상품 패널 입력 UI를 공유합니다. 신규 등록은 `isNew=true` 상태, 기존 상품 조회/수정은 상품 데이터가 채워진 상태로 구분합니다.
- DLG-P001 / `docs4/V1/D05-상품관리/상품관리.md:496` 상품등록모달은 별도 컴포넌트명이 아니라, 공유 상품 패널의 신규 등록 상태를 가리키는 기획 식별자입니다.
- 별도 상품 수정 전용 모달은 사용하지 않습니다. 기존 상품 수정은 SCR-P003 / `docs4/V1/D05-상품관리/상품관리.md:300` 상품 상세/수정 패널에서 같은 패널 UI로 바로 처리합니다.
- 전 지점 배포부터 시즌가격 등록/수정까지 모든 다이얼로그는 P001~P023 번호대를 사용하되, 문서 참조 시에는 항상 DLG-P001 / `docs4/V1/D05-상품관리/상품관리.md:496` 상품등록모달 같은 전체 ID로 표기합니다.




## SCR-P001 상품 관리

> 연결 기능: PRD-01
### 1. 목적

센터에서 판매하는 모든 상품(회원권, 수강권, 락커, 운동복 대여비, 일반 상품)을 한눈에 파악하고 관리하는 메인 화면입니다. 상품별 가격, 이용 기간, 요일·시간 제한, 판매 현황을 목록으로 조회하고, 우측 패널을 통해 상세 정보를 확인하거나 수정할 수 있습니다. 여기서 `운동복`은 실제 수량을 차감하는 재고 품목이 아니라 회원에게 청구하는 대여비/상품 비용입니다.

### 상단 통계 카드

| 카드 | 설명 |
|------|------|
| 전체 상품 수 | 센터에 등록된 상품 총 개수 |
| 활성 상품 수 | 현재 판매 중인 상품 수 |
| 비활성 상품 수 | 판매 중지된 상품 수 |

### 3단계 필터

- **상품 화면 필터 분류 6개**: 전체 / 회원권 / 수강권 / 락커 / 운동복 / 일반. `전체`는 화면 필터 전용 옵션이며 실제 상품 마스터 대분류에는 저장하지 않습니다.

- **상품 분류 6개**: 전체 / 회원권 / 수강권 / 락커 / 운동복 / 일반. 화면에서 선택 가능한 분류 집합이며, `전체`는 조회용 분류입니다.

| 단계 | 필터 유형 | 옵션 |
|------|-----------|------|
| 1단계 | 상품 대분류 탭 | 전체 / 회원권 / 수강권 / 락커 / 운동복 / 일반 |
| 2단계 | 세부 유형 필터 | 1단계 선택값에 따라 동적 표시. 수강권 선택 시 PT / GX / 골프 / 기타 수강권, GX 선택 시 요가 / 필라테스 / 스피닝 / 줌바 / GX 기타 |

- 우측 상단에 상품명 검색 입력창 제공

### 상품 마스터 분류 모델

상품 마스터 저장 카테고리는 회원권 / 수강권 / 락커 / 운동복 대여비 / 일반 5개입니다.

- **상품 마스터 저장 카테고리 5개**: 회원권 / 수강권 / 락커 / 운동복 대여비 / 일반. 상품 저장값은 이 5개만 사용하며, `전체`는 목록 조회 필터에서만 사용합니다.

| 단계 | 필드 | 값 |
|------|------|------|
| 1단계 | 상품 대분류 `category` | `MEMBERSHIP`(회원권) / `LESSON_PASS`(수강권) / `LOCKER`(락커) / `WEAR`(운동복 대여비, 재고 수량 차감 없음) / `GENERAL`(일반) |

| 대분류 | 세부종목 |
|------|------|
| GX | 요가 / 필라테스 / 스피닝 / 줌바 / GX 기타 |

- `productGroupId`가 있는 상품은 연결된 `product_groups.name`을 목록 분류 필터와 상품명 배지에 우선 표시합니다.

- `productGroupId`가 없는 상품은 `category`의 한글 표시명(회원권/수강권/락커/운동복/일반)을 fallback으로 사용합니다.

### 상품 목록 테이블

패널이 닫혀 있을 때 전체 컬럼이 표시됩니다.

| 컬럼 | 내용 |
|------|------|
| 상품명 | 유형 배지 + 패키지 배지 + 운영정책 배지(단일지점/법인/홈&오피스/주말/통합) + 상품명 |
| 대분류 | 상품 대분류 |
| 이용 가능 지점 | 단일/복수/전체 지점 요약 |
| 정산 정책 | 이용지점 기준 / 판매지점 기준 / 지정 정산지점 |
| 현금가 | 현금 결제 시 가격 |
| 카드가 | 카드 결제 시 가격 |
| 기간 | 이용 가능 기간 (일 단위) |
| 횟수 | 이용 가능 횟수 |
| 요일 (월~일) | 이용 가능 요일 표시 (체크/엑스) |
| 시간 | 이용 가능 시간대 |
| 홀딩 | 홀딩(일시정지) 가능 여부 |
| 양도 | 양도 가능 여부 |
| 포인트 | 포인트 적립 여부 |
| 판매 | 해당 상품 판매 건수 |
| 상태 | 사용 / 미사용 |

> 우측 상세 패널이 열리면 요일/시간/홀딩/양도/포인트/판매 컬럼은 자동으로 숨겨지고 핵심 컬럼만 표시됩니다.

### 분류 관리 탭

상품 목록 탭과 분류 관리 탭을 전환할 수 있습니다.

- 분류 등록 폼: 분류명, 정렬 순서, 활성 여부 입력

- 분류 목록 테이블: 분류명 / 정렬 순서 / 상태 / 수정·삭제 버튼

### 3. 액션 / 흐름

1. **상품 조회**: 화면 진입 시 센터 전체 상품 목록 표시 → 필터/검색으로 원하는 상품 찾기

2. **상품 상세 보기**: 목록에서 상품 행 클릭 → 우측에 상세 패널이 슬라이드로 열림

3. **상품 등록**: 상단 "+ 상품 등록" 버튼 클릭 → 우측 패널이 등록 모드로 열림

4. **분류 관리**: "분류 관리" 탭 클릭 → 상품 분류 등록/수정/삭제

5. **전 지점 배포** (슈퍼관리자): "전 지점 배포" 버튼 → 배포 모달에서 상품·지점 선택 후 일괄 배포

---

### 4. 핵심 상태

| 상태 | 조건 | 표시 내용 |
|------|------|-----------|
| 로딩 중 | 상품 데이터 불러오는 중 | 목록 영역 로딩 표시 |
| 빈 상태 | 등록된 상품 없음 또는 필터 결과 없음 | "상품이 없습니다." 안내 + 상품 등록 링크(권한 있는 경우) |
| 정상 | 상품 목록 존재 | 상품 목록 테이블 표시 |
| 패널 열림 | 상품 행 클릭 또는 등록 버튼 클릭 | 목록 55% + 우측 패널 45% 분할 표시 |
| 패널 닫힘 | 패널 X 버튼 클릭, 저장/삭제 완료 | 목록 100% 표시 |

### 6. 필수 예외처리

| 케이스 | 표시 | 후속 조치 |
|---|---|---|
| 상품 0건(신규 지점) | 빈 상태 + [+ 상품 등록] CTA | 등록 안내 |
| 검색·필터 결과 0건 | 빈 안내 + [필터 초기화] | 조건 초기화 |
| 상품명 중복(동일 지점·대분류) | 인라인 "이미 등록된 상품명입니다" | 저장 차단 |
| 카드가 < 현금가 | 노랑 경고 배지 | 진행 가능 |
| 카드가 > 현금가 | 빨강 인라인 + 저장 차단 | 가격 재확인 |
| 상품그룹 삭제 시도(연결 상품 존재) | 모달 + 연결 상품 {개수}개 | [그룹 이동] / [취소] |
| 전 지점 배포 일부 실패 | 진행률 모달 + 실패 지점 분리 | 실패 지점만 재시도 |
| 엑셀 1만 건 초과 | "비동기 모드 전환" 토스트 | 알림 센터 통보 |
| 폐점 지점 등록 시도 | "폐점 지점은 신규 등록 불가" | 차단 |
| 본사 통합 모드 등록 시도 | "지점을 먼저 선택해주세요" | 지점 선택 강조 |
| 동시 수정 충돌(다른 매니저) | "다른 사용자가 먼저 저장했어요" | [최신 정보 보기] / [덮어쓰기] |
| 패키지 구성 상품 삭제 시도 | "패키지에 사용 중입니다" + 패키지 {개수}개 | 패키지 분리 후 삭제 |
| 비활성 그룹 신규 상품 등록 | "비활성 그룹은 신규 등록 불가" | 활성 전환 안내 |
| 키오스크 노출 ON + 비활성 상품 | 자동 OFF + 안내 토스트 | 키오스크 자동 제외 |
| 분류 관리 정렬 충돌(동일 순서) | 등록일 ASC로 자동 보조 정렬 | 인라인 안내 |

| 요소 | 설명 |
|---|---|
| 제목 | `상품등록` |

### 상품 구분

| 항목 | 필수 | 설명 |
|---|---|---|
| 사용인원 | 수강권 선택 시 | 기본 1명 |
| 1단계 | 수강권 선택 시 | PT / GX / 골프 / 기타 수강권 |
| 2단계 | GX 선택 시 필수 | 요가 / 필라테스 / 스피닝 / 줌바 / GX 기타 |

| 항목 | 필수 | 설명 |
|---|---|---|

| 태그 | 선택 | 검색·분류용 태그 |
| 설명 | 선택 | 상품 설명 |

### 요일 / 시간설정

| 항목 | 설명 |
|---|---|
| 월~일 체크박스 | 체크한 요일만 이용 가능 |
| 시작/종료 시간 | 체크된 요일에서만 활성화. 기본 09:00~18:00 |

## DLG-P002-전지점배포 전 지점 배포 확인

> 연결 기능: 없음 (DLG-P002는 docs4 내부에서 독립 보조 화면/다이얼로그로 관리)
### 1. 목적

슈퍼관리자가 현재 센터에 등록된 상품을 다른 지점 전체 또는 선택한 지점에 동시에 배포하는 모달입니다. 이미 같은 이름의 상품이 있는 지점은 자동으로 건너뛰어 중복 등록을 방지합니다.

### 배포할 상품 선택 영역

- 전체 상품 목록을 체크박스로 표시

- "전체 선택" 버튼 제공

- 선택된 상품 수 표시 (예: "3개 선택됨")

### 대상 지점 선택 영역

- "전체 지점" 체크박스 (기본 선택)

- 전체 지점 체크 해제 시: 개별 지점 체크박스 목록 표시

- 현재 로그인 지점은 목록에서 제외

하단 버튼: **취소** / **배포하기**

---

### 3. 액션 / 흐름

1. 배포할 상품 선택 (1개 이상 필수)

2. 대상 지점 선택 (1개 이상 필수)

3. "배포하기" 버튼 클릭

   - 상품 미선택 시: "배포할 상품을 선택하세요." 안내

   - 지점 미선택 시: "배포할 지점을 선택하세요." 안내

   - 정상 진행 시: 배포 최종 확인 단계로 이동

4. 최종 확인에서 승인 시: 지점별 배포 처리

   - 이미 같은 이름의 상품이 있는 지점은 건너뜀

   - 완료 후 "N건 추가, N건 건너뜀" 결과 메시지 표시

### 4. 핵심 상태

| 상태 | 설명 |
|---|---|
| 기본 표시 | 부모 화면에서 선택한 대상의 최신 값을 불러와 표시합니다. |
| 입력/수정 중 | 필수값 누락, 형식 오류, 권한 제한을 인라인 메시지로 표시합니다. |
| 저장/처리 중 | 주요 버튼을 비활성화하고 중복 요청을 막습니다. |
| 완료 | 저장 또는 처리 성공 후 부모 화면 데이터를 갱신하고 다이얼로그를 닫습니다. |
| 실패 | 실패 사유를 토스트 또는 인라인 메시지로 안내하고 입력값은 유지합니다. |

### 5. 권한

| 계정 | 동작 기준 |
|---|---|
| Owner(지점장) / manager | 부모 화면에서 허용된 경우 조회·입력·저장할 수 있습니다. |
| trainer / fc / staff | 담당 업무 범위에 포함된 데이터만 조회하며, 수정 권한은 부모 화면 정책을 따릅니다. |

| 권한 없음 | 다이얼로그 진입을 차단하고 부모 화면에서 권한 안내를 표시합니다. |

### 7. 연동 / 자동화 참조

| 연계 대상 | 트리거 | 영향 |
|---|---|---|
| Cron - 비활성 만료 회원 정리 | 매일 04:00 | 비활성 상품 보유 회원 만료 시 정리 |
