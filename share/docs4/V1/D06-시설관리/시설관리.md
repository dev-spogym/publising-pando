# D06-시설관리 시설관리

> 개발사용 정리 문서입니다. 이 파일만으로 해당 도메인의 화면·다이얼로그 구현 기준을 확인할 수 있도록 재구성했습니다.




## SCR-050 락커 관리

> 연결 기능: 없음 (V1 범위 직접 연결 기능 없음; 공통/보조 화면 또는 V2 기능은 V2 문서에서 관리)
### 2. UI 구성

- **현황 요약 카드**: 전체 락커 수 / 사용중 수 / 만료 임박 수 / 고장 수를 화면 상단에 4개 카드로 표시합니다.

- **자동 회수 정책 상태 배지**: D09 SCR-080A / `docs4/V1/D09-설정관리/설정관리.md:141` 지점 자동화 적용에서 현재 적용 중인 만료 경과 자동 회수, 이용권 만료 이벤트 자동 회수의 ON/OFF 상태를 읽기 전용으로 표시합니다.

- **구역 탭**: A구역 · B구역 · C구역으로 탭을 분리하여 원하는 구역료 집중해서 볼 수 있습니다.

- **보기 전환**: 박스 뷰 / 리스트 뷰를 전환할 수 있습니다. 박스 뷰는 실제 배치와 상태 시각화에, 리스트 뷰는 대량 검색과 정렬에 적합합니다.

- **상태 필터 칩**: 전체 / 사용중 / 미배정 / 만료 임박 / 고장 필터를 칩 형태로 제공하고 각 상태 건수를 함께 표시합니다.

- **락커 배치 그리드**: 실제 락커 배치와 동일한 형태의 격자 화면으로 각 락커를 색상으로 구분합니다.

  - 각 셀에 락커 번호, 회원명(배정 시), 남은 일수를 표시합니다.

  - 셀에 마우스를 올리면 기록 · 이동 · 회수 · 배정 · 고장 처리 버튼이 나타납니다.

- **리스트 뷰 테이블**: 락커 번호 / 구역 / 상태 / 배정 회원 / 상품명 / 시작일 / 만료일 / 남은 일수 / 비고 / 액션 컬럼을 제공합니다.

- **색상 범례**: 그리드 하단에 상태별 색상 의미를 안내합니다.

- **검색창**: 락커 번호 또는 배정된 회원명으로 빠르게 해당 락커를 찾을 수 있습니다.

- **미배정 우측 패널**: 빈 락커를 클릭하면 우측에서 회원 검색 패널이 열리고, 회원 검색·추천 후보·만료일 입력 후 바로 배정할 수 있습니다.

- **만료 임박 일괄 해제**: 탭 우측에 버튼을 클릭하면 만료 임박 락커를 선택해 한 번에 회수할 수 있습니다.

### 3. 액션 / 흐름

1. 화면 진입 시 현황 요약 카드와 A구역 그리드가 기본으로 표시됩니다.

2. 구역 탭·상태 필터·보기 전환을 조합해 현재 작업 문맥(시각화/검색/배정)을 선택합니다.

3. 검색창에 락커 번호·회원명을 입력해 원하는 락커를 찾습니다.

4. 박스 뷰에서는 셀 hover로 액션 버튼을 보고, 리스트 뷰에서는 행 액션 메뉴에서 동일 동작을 수행합니다.

5. 빈 락커: [배정] 버튼 또는 셀 클릭 → 우측 회원 검색 패널 오픈 → 회원 선택 → 만료일 입력 → 배정 완료.

8. 일괄 배정: [일괄 배정] 버튼 클릭 → 빈 락커를 클릭해 선택 → 회원·만료일 입력 → 한 번에 배정.
