# 개발사 질의 방지 체크리스트

기능명, 배지, 알림, 상태, 대시보드, 연동 문장을 작성하거나 수정할 때 아래 항목을 반드시 채웁니다.

| 항목 | 반드시 명시할 것 |
|---|---|
| 배지 | 표시 화면, 표시 위치, 카운트 기준, 0건 표시 여부, 클릭 시 이동 |
| 알림 | 발생 조건, 수신 권한, 표시 위치, 클릭 시 이동, 읽음/해제 조건 |
| 상태값 | 전체 enum, 전환 조건, 필터 포함 여부, 완료/대기 집계 기준 |
| 대시보드 | SCR-090/SCR-094/SCR-095/SCR-098/SCR-101 중 정확한 대상 |
| 권한 | 역할별 조회 범위, 버튼 노출, 실행 가능 여부, 예외 권한 |
| 정책값 | 본사/지점/고객사 중 설정 주체, 기본값, 변경 권한, 적용 범위 |
| 연동 | 원천 화면, 대상 화면, 트리거, 후속 상태 변화, 실패 시 처리 |
